### 문제 이해

직전 문제인 A+B(4)에 조건문을 사용하여 탈출하게 한다.

### 소스 코드 (Python)

```python
if __name__ == "__main__":
    while True:
        a, b = map(int, input().split())
        if(a==0 and b==0):
            break;
        print(a+b)
```

### 소스 코드 (Java)

```java
import java.util.Scanner;

public class java_10952 {    // 채점 시 Class 명을 'Main'으로 변경

    public static void main(String[] args) {

        Scanner scan = new Scanner(System.in);
        int a,b;

        while(true){
            a = scan.nextInt();
            b = scan.nextInt();
            if(a==0 && b==0) break;
            System.out.println(a+b);
            //scan.nextLine();
        }

    }

}
```

### 소스 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main(void){
    ios_base :: sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    int a;
    int b;
    while(true){
        cin>>a>>b;
        if(a==0 && b==0) break;
        cout<<a+b<<"\n";
    }
    //system("pause");
    return 0;
}
```
